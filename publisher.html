ℹ️ كيفية الحصول على مفاتيح API
📘 فيسبوك / إنستغرام:
1. افتح developers.facebook.com
2. أنشئ تطبيقاً جديداً من نوع Business
3. أضف منتج Graph API ← احصل على Page Access Token
4. في Graph Explorer ← اختر صفحتك ← انسخ الـ Token
💬 واتساب Business:
1. افتح business.whatsapp.com
2. سجّل حساباً للأعمال ← احصل على Phone Number ID
3. من لوحة التحكم ← API Access ← انسخ Access Token
✈️ تيليغرام:
1. تحدث مع @BotFather على تيليغرام
2. أرسل /newbot ← اختر اسماً ← احصل على Bot Token
3. أضف البوت لقناتك ← احصل على Channel ID
